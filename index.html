ğŸ“š Tech Glossary

HTML
HTML is a Markup Language used to create and structure web pages. It uses markup tags to define how content like text, images, and links should be displayed in a web browser. HTML is the foundation of most websites, providing the basic skeleton and content structure for a page.
CSS
CSS (Cascading Style Sheet) is a stylesheet language used to describe the presentation of a web page written in HTML or XML. It dictates how HTML elements are displayed on screen, paper, or in other media, controlling aspects such as colors, fonts, layout, spacing, and responsive behavior for different devices. By separating the structure of a document (HTML) from its visual presentation (CSS), CSS enables web developers to create visually appealing and consistent designs across multiple pages, making websites easier to maintain and update.
JS
JavaScript is a versatile, high-level, and dynamically typed programming language primarily known for bringing interactivity to web pages. It is a core technology of the World Wide Web, alongside HTML and CSS, enabling dynamic content updates, user interaction handling, and complex web application development. Initially conceived for client-side scripting in web browsers, its utility has expanded significantly with environments like Node.js, allowing server-side development and a wide range of non-browser applications.
